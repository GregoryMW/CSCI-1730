CSCI 1730 Project 4

Name of program: yosh.c

This program is a user-made shell within the UNIX enviroment. This shell allows the user to use various built-in commands for certain tasks: ls, cd, etc. [WARNING] At the time of submission, this
project was incomplete, and thus does not have full functionality.

CODER: Gregory Woolsey

CREDIT/Contributions: parse.h & shell.c given in the Project 4 materials

How to compile:
    gcc -g -Wall -o yosh yosh.c parse.c

OR use the included Makefile
    make

How to run:
    yosh
    Can then use yosh shell similar to other UNIX-based shells
